# Breast Cancer Classification using KNN

This repository contains a tutorial and code for classifying breast cancer using the K-Nearest Neighbors (KNN) algorithm.

## Files

- tutorial.pdf – Step-by-step tutorial explaining KNN and its application to breast cancer classification.
- Breast_Cancer_KNN.ipynb – Jupyter notebook with the code, data analysis, and plots.
- data – Optional folder for datasets.
- 

## How to Use

1. Open the Jupyter notebook (`Breast_Cancer_KNN.ipynb`) to run the code.
2. Read the tutorial (`tutorial.pdf`) for explanations and insights.
3. All necessary references are included in the PDF.
